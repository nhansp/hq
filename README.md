# Dự án: Quản lý hàng đợi bệnh viện
## Nhóm 4 - Cấu trúc dữ liệu và giải thuật (DASH100)

# Yêu cầu hệ thống

Python 3, pip, git

# Chạy chương trình

* Clone repo:

    ```bash
    git clone https://github.com/nhansp/hq
    ```

* Chạy:

    * Linux/MacOS:
    ```bash
        ./run.sh
    ```

    * Windows:
    ```bash
        python3 app.py
    ```
